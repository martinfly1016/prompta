### GitHub PAT scope 教训

PAT 缺 `workflow` scope 不能 push `.github/workflows/*.yml` — Fine-grained PAT 配置页面里叫「Workflows」(不是 classic PAT 的 `workflow` scope)，需在 Permissions → Add permissions → Workflows → Read and write。

### gh CLI 设置

`brew install gh` 后需要登录。试过 `security find-internet-password -s github.com -w | gh auth login --with-token` 但 keychain 里的 token 不被 gh API 接受（401）。最终 fallback 到 user 自己手动 `gh auth login` 或浏览器 Run workflow 按钮。

未来命令行直接监控 workflow 需要 `gh auth login` 一次。
