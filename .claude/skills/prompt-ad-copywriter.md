# X平台Prompt广告文案生成器

## 功能描述
这个Skill帮助你为Prompta平台上的Prompt快速生成高质量的X平台（Twitter）广告文案，包含日文和中文版本。

## 使用方法

### 第一步：获取Prompt页面信息
请提供Prompta Prompt的完整URL，格式如下：
```
https://www.prompta.jp/prompt/[PROMPT_ID]
```

### 第二步：文案生成流程
系统会自动：
1. 访问并分析Prompt页面内容
2. 提取核心信息（标题、描述、功能特点）
3. 生成吸引人的日文文案
4. 生成对应的中文翻译
5. 添加相关hashtags

### 第三步：文案结构
生成的文案包含以下部分：

#### 日文版本（Japanese）
- 开场吸引句（使用emoji）
- 核心功能描述（3-4个bullet points）
- 行动号召（CTA）
- Hashtags

#### 中文版本（Chinese）
- 开场吸引句（使用emoji）
- 核心功能描述（3-4个bullet points）
- 行动号召（CTA）
- Hashtags

## 文案特点
- ✨ 使用emoji增加视觉吸引力
- 📱 针对X平台特性优化（字数、格式、hashtags）
- 🎯 突出Prompt的核心价值
- 🌐 日英双语，便于不同市场推广
- 📸 配合效果图使用效果最佳

## 示例输出格式

### 日文版
```
✨[开场句]✨

[功能描述]

📍 特点1
📍 特点2
📍 特点3

[行动号召]

#tag1 #tag2 #tag3
```

### 中文版
```
✨[开场句]✨

[功能描述]

📍 特点1
📍 特点2
📍 特点3

[行动号召]

#tag1 #tag2 #tag3
```

## 注意事项
- 文案应简洁有力，适合社交媒体传播
- 使用emoji但不过度
- 包含明确的行动号召
- Hashtags应该相关且流行
- 日文应自然流畅，中文翻译要准确传达意思
